# Investigacion-DMD

### Descripción
Casos de la Investigación Documental “Aplicación del Datawarehouse y el Análisis Multidimensional en la Toma de Decisiones Empresariales”

## Tecnologías Utilizadas
* Visual Studio 2022 -> https://learn.microsoft.com/en-us/visualstudio/install/install-visual-studio?view=vs-2022
* SQL Server Management Studio -> https://learn.microsoft.com/en-us/ssms/install/install
* Microsoft Excel

## Extensiones de Visual Studio 2022
* Integrations Service
* Analysis Service
* Reporting Service
* OLE DB -> https://learn.microsoft.com/en-us/sql/connect/oledb/download-oledb-driver-for-sql-server?view=sql-server-ver17

## Despliegue
Deberá extraer dos veces el archivo descargado.

Abrir Visual Studio 2022 y dar click al botón "Open a project or solution"

Una vez abierto el buscardor de archivos, deberá seleccionar el archivo y la solución del caso que desea abrir

Al abrir el proyecto/solución se mostrará el siguiente mensaje "This project was downloaded from the web", deberá dar clcik en el botón "Trust and Continue" y así sucesivamente con los demás casos
